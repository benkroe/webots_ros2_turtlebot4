# webots_ros2_turtlebot4
This fork adds the meshes for the turtlebot4 directly to the package and shared folder. Therefore it adds the capability to use the package with ros2 in a Virtual machine (UTM) and webots native on macOS. It works like the following example with the epuck and pre-configured image:
https://docs.ros.org/en/foxy/Tutorials/Advanced/Simulators/Webots/Installation-MacOS.html
